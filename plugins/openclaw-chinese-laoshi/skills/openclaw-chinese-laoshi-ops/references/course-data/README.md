# Course Data

Sanitized, small learner data bundled with the public skill.

Included files:

- `course-index.md`: public course index.
- `lessons-bundle.json`: public lesson JSON bundle.
- `lessons-bundle.md`: public lesson Markdown bundle.
- `lesson-plans/`: generated lesson plans.
- `hsk/`: generated HSK-style practice payloads.
- `roleplays/`: generated roleplay scenarios and daily sprints.
- `hsk-training.json`: HSK training manifest.
- `roleplay-training.json`: roleplay training manifest.

Excluded by design: raw media, raw transcripts, Drive IDs, local paths,
source segment indexes, review-control metadata, and secrets.
